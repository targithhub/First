What is Git?

Git is use for version control. 1. Track History 2. Work Together

How to use Git

Step 1 - Create account on Githhub and create repository and clone it our syestem folder.


Step 2 - Install Git in our computer and configure it.


Step 3 - Now we can create a html file


Step 5 - Run command for commit like
git commit -m 'message'


Step 5 - Command for Upload file on Githhub
git push origin filename


Step 6 - To create branch
git branch branchname


Step 7 - To switch branch
git branch branchname


Step 8 - To check commit History
git log


Step 9 - To merge two branch
git merge branchname


Step 10 - To get files from Githhub
git pull branchname remote URL

remote url means Githhub URL
